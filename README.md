# Projet de Conversion de Devises et Gestion de Bibliothèque

Ce projet comprend plusieurs exercices pratiques et tests unitaires pour une application de conversion de devises et un système de gestion de bibliothèque. Il utilise JUnit pour les tests unitaires, Maven pour la gestion des dépendances et l'automatisation des tests, ainsi que JaCoCo pour la couverture de code.
